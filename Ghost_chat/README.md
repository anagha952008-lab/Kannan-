**Ghost-Chat** എന്നത് പൂർണ്ണമായും ടെർമിനലിൽ പ്രവർത്തിക്കുന്ന അതീവ സുരക്ഷിതമായ ഒരു End-to-End Encrypted (E2EE) ചാറ്റ് സിസ്റ്റം ആണ്. മിലിട്ടറി ഗ്രേഡ് സുരക്ഷ ഉറപ്പാക്കാൻ ഇതിൽ **AES-256** എൻക്രിപ്ഷൻ സാങ്കേതികവിദ്യയാണ് ഉപയോഗിച്ചിരിക്കുന്നത്. നിങ്ങൾ ചാറ്റ് റൂമിൽ നിന്നും സ്വയം എക്സിറ്റ് ആകുന്നത് വരെ കണക്ഷൻ ബാക്ക്ഗ്രൗണ്ടിൽ എപ്പോഴും നിലനിൽക്കും.

---

## ✨ പ്രധാന ഫീച്ചറുകൾ (Key Features)

* **End-to-End Encryption:** സന്ദേശങ്ങൾ സെർവറിലേക്ക് പോകുന്നതിന് മുൻപ് തന്നെ എൻക്രിപ്റ്റ് ചെയ്യപ്പെടുന്നു.
* **Persistent Connection:** യൂസർ സ്വന്തമായി വിട്ടുപോകുന്നത് വരെ 24 മണിക്കൂറും കണക്ഷൻ നിലനിൽക്കും.
* **No Logs Policy:** സെർവറിൽ ഒരു തരത്തിലുള്ള ചാറ്റ് ഹിസ്റ്ററിയോ ലോഗുകളോ സൂക്ഷിക്കുന്നില്ല.
* **Modern Terminal UI:** മനോഹരമായ കളറുകളും ASCII ആർട്ടും നിറഞ്ഞ ഇന്റർഫേസ്.

## 🚀 ഇൻസ്റ്റലേഷൻ രീതി (Installation Guide)

ഈ ടൂൾ നിങ്ങളുടെ സിസ്റ്റത്തിൽ ഇൻസ്റ്റാൾ ചെയ്ത് ഉപയോഗിക്കാൻ താഴെ പറയുന്ന കമാൻഡുകൾ നിങ്ങളുടെ ടെർമിനലിൽ ഒന്നൊന്നായി അടിക്കുക:

```bash
# 1. പ്രോജക്റ്റ് ഗിറ്റ്ഹബ്ബിൽ നിന്നും ഡൗൺലോഡ് ചെയ്യുക

# 2. പ്രോജക്റ്റ് ഫോൾഡറിലേക്ക് പ്രവേശിക്കുക

# 3. ആവശ്യമായ ലൈബ്രറികൾ ഇൻസ്റ്റാൾ ചെയ്യുക

# ​4. സെർവർ ഓൺ ചെയ്യാൻ (Start Server)
python3 chat_server.py

#5. ചാറ്റിൽ ജോയിൻ ചെയ്യാൻ (Start Client)
​python3 chat_client.py

#7.ടൂളുകൾ ഡൗൺലോഡ് ചെയ്യാൻ (Download Tool)
