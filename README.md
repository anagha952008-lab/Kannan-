## 🎯 ഈ ടൂളിന്റെ ലക്ഷ്യം (What is GHOST-CHAT?)
സൈബർ സെക്യൂരിറ്റി വിദ്യാർത്ഥികൾക്കും എത്തിക്കൽ ഹാക്കർമാർക്കും പരസ്പരം സുരക്ഷിതമായി സംസാരിക്കാനും, അവർ വികസിപ്പിക്കുന്ന സെക്യൂരിറ്റി ടൂളുകളുടെ ലിങ്കുകൾ പരസ്പരം കൈമാറാനുമുള്ള ഒരു **End-to-End Encrypted Cloud Platform** ആണിത്.

### 🔥 പ്രധാന പ്രത്യേകതകൾ:
* **പാസ്‌വേഡ് രഹിത കണക്ഷൻ:** ഡാറ്റാബേസ് പാസ്‌വേഡ് ഹാക്കർമാർ മോഷ്ടിക്കാതിരിക്കാൻ പൂർണ്ണമായും രഹസ്യ ഫയൽ സിസ്റ്റം ഉപയോഗിക്കുന്നു.
* **തത്സമയ ചാറ്റിംഗ് (Realtime):** ലൂപ്പുകൾ ഇല്ലാതെ ഗൂഗിൾ ഫയർബേസ് ക്ലൗഡ് വഴി മെസ്സേജുകൾ അപ്പപ്പോൾ സ്ക്രീനിൽ വരുന്നു.
* **അതീവ സുരക്ഷ (Fernet AES Encryption):** നിങ്ങളുടെ മെസ്സേജുകൾ ഡാറ്റാബേസിൽ പോകുന്നതിന് മുൻപ് തന്നെ ലോക്ക് ചെയ്യപ്പെടുന്നു. ലിങ്ക് ചോർന്നാൽ പോലും ആർക്കും മെസ്സേജ് വായിക്കാൻ കഴിയില്ല!

## 🎯 ടൂളിന്റെ പ്രധാന ലക്ഷ്യം (Features)

സൈബർ സെക്യൂരിറ്റി വിദ്യാർത്ഥികൾക്കും എത്തിക്കൽ ഹാക്കർമാർക്കും തങ്ങളുടെ നെറ്റ്‌വർക്കിൽ സുരക്ഷിതമായി തത്സമയം സംസാരിക്കാനും ടൂളുകൾ പരസ്പരം പങ്കുവെക്കാനും ഈ പ്ലാറ്റ്‌ഫോം സഹായിക്കുന്നു.

* **🔑 Password-Less Connection:** ഡാറ്റാബേസ് പാസ്‌വേഡ് പൂർണ്ണമായും ഒളിപ്പിച്ചു വെച്ചുകൊണ്ട് സുരക്ഷിതമായ രഹസ്യ ഫയൽ സിസ്റ്റം (`.env`) ഉപയോഗിക്കുന്നു.
* **⚡ Realtime Synchronization:** ഗൂഗിൾ ഫയർബേസിന്റെ കരുത്തിൽ ലൂപ്പുകൾ ഇല്ലാതെ തന്നെ മെസ്സേജുകൾ തത്സമയം സ്ക്രീനിൽ തെളിയുന്നു.
* **🛡️ Fernet AES Encryption:** ഡാറ്റാബേസിലേക്ക് പോകുന്നതിന് മുൻപ് തന്നെ മെസ്സേജുകൾ ശക്തമായി ലോക്ക് ചെയ്യപ്പെടുന്നു. ലിങ്ക് ചോർന്നാൽ പോലും മറ്റാർക്കും ഇത് വായിക്കാൻ കഴിയില്ല!
* **🎨 Rich Terminal GUI:** ടെർമിനലിൽ തന്നെ മനോഹരമായ ടേബിളുകളും കളർഫുൾ ടെക്സ്റ്റുകളും നൽകുന്ന ഇന്റർഫേസ്.
